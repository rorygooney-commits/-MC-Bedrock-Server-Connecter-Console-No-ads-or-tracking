# -MC-Bedrock-Server-Connecter-Console-No-ads-or-tracking
A lightweight, LAN-based Minecraft Bedrock server connector for consoles. Allows PS4, PS5, Xbox, and Switch players to join any Bedrock server without DNS tricks or router configuration — inspired by BedrockConnect. 100% free, no ads, no tracking, no telemetry.

## ✅ Features

- Broadcasts a LAN world visible in the Minecraft Friends tab
- Proxies UDP traffic to any Bedrock server (IP + port)
- Lightweight — runs on any machine on your local network
- No router changes or DNS configuration needed
- Supports multiple servers via config file
- Cross-platform: runs on Windows, macOS, and Linux
- 🚫 No ads
- 🚫 No tracking
- 🚫 No telemetry
- 🚫 No accounts required
- 🚫 No data collection of any kind
- ✅ Fully open source — see exactly what runs on your machine
REQUIRES THAT THE DEVICE YOU ARE USING TO BROADCAST MUST BE ON THE SAME NETWORK AS THE DEVICE YOU ARE PLAYING ON KEEP IT ON WHILE YOU PLAY
